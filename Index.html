Proyectos

Proyecto 1: Creación de una web usando HTML5

Creación de mi primera página web utilizando las etiquetas y estructura semántica de HTML5, sentando las bases para el desarrollo web.

Proyecto 2: Maquetación y Estilo con CSS

Aplicación de hojas de estilo CSS para mejorar la presentación de páginas HTML. Este proyecto cubre el posicionamiento de elementos, el uso de pseudoclases para interactividad y la manipulación del modelo de caja (padding, margin, border) para crear diseños complejos y atractivos.

Proyecto 3: Implementación de Servidor FTP

Configuración y administración de un servidor FTP en entornos Windows Server y Ubuntu Server. Incluye la instalación del servicio, la gestión de permisos y usuarios, y el uso de comandos de cliente para la transferencia segura de archivos.

Proyecto 4: Instalación y Configuración de Sistemas Operativos

Proceso completo de instalación y configuración inicial de los sistemas operativos Windows y Ubuntu. Abarca desde la partición del disco hasta la configuración de red, usuarios y software esencial post-instalación.
